# Usage of ./m

```cmd
./m Folder_name [subfolder_name1 subfolder_name2 ...]
```
## Example 
```cmd
./m abc100
```
will generate: 

abc100 and subfolders: A B C D E F G H I J

```cmd
./m abc100 A B C
```
will generate: 

abc100 and subfolders: A B C
